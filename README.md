# Alkesh Gupta — Personal Brand Portfolio

A modern, responsive personal branding website built with **Next.js 16**, **React 19**, **Tailwind CSS v4**, and **Framer Motion**. The website showcases Alkesh Gupta's journey, philosophy, featured brands, testimonials, founder story, and vision through a clean, premium user experience with smooth animations.

---

## ✨ Features

* Modern and fully responsive design
* Smooth page animations using Framer Motion
* Modular component architecture
* Optimized image handling with Next.js
* Clean typography and premium visual styling
* Founder story & brand philosophy sections
* Featured brands showcase
* Testimonials section
* Responsive navigation and footer
* Built using the latest React and Next.js versions

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/your-username/your-repository.git
```

```bash
cd your-repository
```

---

### Install dependencies

Using npm

```bash
npm install
```

Using pnpm

```bash
pnpm install
```

Using yarn

```bash
yarn
```

---

### Run the development server

```bash
npm run dev
```

or

```bash
pnpm dev
```

Visit:

```
http://localhost:3000
```
